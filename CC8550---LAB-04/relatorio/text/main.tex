\section{Exercício 6 --- Teste Completo (Integrador)}

\subsection{Código}

\begin{lstlisting}
def analisar(numeros):
    total = 0
    for n in numeros:
        if n > 0 and n % 2 == 0:
            total += n
        elif n < 0:
            total -= 1
        else:
            continue
    if total > 10:
        return "Acima"
    return "Abaixo"
\end{lstlisting}

\begin{figure}[H]
\centering
\includegraphics[width=0.72\textwidth]{ex6_gfc}
\caption{GFC Exercício 6}
\end{figure}

$V(G) = 4$. C0/C1: cobrir total+=n, total-=1, continue, return Acima/Abaixo. Casos (resumo): []$\to$Abaixo; [6]$\to$Abaixo; [6,8]$\to$Acima; [1],[-2]$\to$Abaixo; [4,3,8],[6,-2,8],[0,6,8]$\to$Acima; [2,4,1]$\to$Abaixo. Def-uso \texttt{total}: def(2,5,7), use(10,11-12).

\section{Exercício 7 --- Fluxo de Dados}

\subsection{Código}

\begin{lstlisting}
def desconto(preco, cliente_vip):
    total = preco                    # linha 2: def(total)
    if cliente_vip:                  # linha 3: use(cliente_vip)
        desconto_valor = preco * 0.2 # linha 4: def(desconto_valor)
        total = preco - desconto_valor # linha 5: def(total)
    if total < 50:                   # linha 6: use(total)
        total = 50                   # linha 7: def(total)
    return total                     # linha 8: use(total)
\end{lstlisting}

\begin{figure}[H]
\centering
\includegraphics[width=0.65\textwidth]{ex7_gfc}
\caption{GFC Exercício 7 (def-uso)}
\end{figure}

$V(G) = 2$. \texttt{total}: def(2,5,7), use(6,8). \texttt{desconto\_valor}: def(4), use(5). 6 du-pairs. Casos: (100,F)$\to$100; (40,F)$\to$50; (100,T)$\to$80; (40,T)$\to$50; (60,F)$\to$60; (150,T)$\to$120. All-Defs e All-Uses cobertos.

\section{Resumo e Conclusões}

Hierarquia: C0 $\leq$ C1 $\leq$ CC $\leq$ caminhos; All-Defs/All-Uses complementam. Resumo: 7 exercícios, V(G) total 17, 39+ casos de teste. Recomendações: atingir no mínimo C1; usar CC em condições compostas; All-Uses em funções críticas.
