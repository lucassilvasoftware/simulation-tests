O que são mutantes mortos?

Mutante morto

É quando a suíte de testes falha ao rodar no código mutado: pelo menos um teste que passava no original passa a falhar. Isso mostra que a alteração (o “defeito” simulado) foi detectada — o mutante é morto pelos testes.

Mutante sobrevivente

É quando todos os testes continuam passando mesmo com o código alterado. A suíte não consegue distinguir o mutante do programa original, o que indica lacuna na cobertura ou nas asserções. O objetivo é ter uma suíte que mate o máximo de mutantes (alto Mutation Score).

Mutantes manuais

Código original vs. versão mutada. Em verde: correto; em vermelho/laranja: mutação. Os testes que falham “matam” o mutante.

Mutante 1 — Depósito zero permitido Morto

Condição alterada de valor <= 0 para valor < 0, permitindo depósito de R$ 0,00. O teste test_nao_permite_deposito_negativo_ou_zero falha e mata o mutante.

Original
if valor <= 0:
    raise ValueError("Valor de depósito deve ser positivo.")
Mutante
if valor < 0:
    raise ValueError("Valor de depósito deve ser positivo.")

Mutante 2 — Não permite sacar todo o saldo Morto

valor > conta.saldo trocado por valor >= conta.saldo. Quem tenta sacar exatamente o saldo recebe erro. O teste test_pode_sacar_todo_saldo mata o mutante.

Original
if valor > conta.saldo:
    raise ValueError("Saldo insuficiente para saque.")
Mutante
if valor >= conta.saldo:
    raise ValueError("Saldo insuficiente para saque.")

Mutante 3 — Transferência na direção errada Morto

Ordem invertida: debita da conta destino e credita na origem. O teste test_transferencia_move_saldo_correto detecta os saldos incorretos e mata o mutante.

Original
sacar(conta_origem, valor)
depositar(conta_destino, valor)
Mutante
sacar(conta_destino, valor)
depositar(conta_origem, valor)

Mutante 4 — Extrato em ordem invertida Morto

list(conta.historico) trocado por list(reversed(conta.historico)). O teste test_extrato_registra_operacoes_na_ordem exige ordem cronológica e mata o mutante.

Original
return list(conta.historico)
Mutante
return list(reversed(conta.historico))

Mutation Score

Mede a qualidade da suíte de testes: qual fração dos mutantes foi morta.

MS = (mutantes mortos) / (mutantes totais − equivalentes) × 100%
No cenário manual: 4 mutantes totais, 4 mortos → MS = 100% (todos detectados).
